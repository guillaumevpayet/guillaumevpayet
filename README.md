# Guillaume Payet — Gameplay & Tools Programmer

📍 New Zealand | 🔗 [Portfolio](https://guillaumepayet.dev/) | 📧 [guillaumevpayet@gmail.com](mailto:guillaumevpayet@gmail.com)

Gameplay and tools programmer with **8+ years** of professional software development experience. I build gameplay AI systems, physics, input pipelines, developer tools, and emulators across Unity, Unreal Engine, Godot, and O3DE.

---

## 🎮 Featured Projects

### [Cruel Paradise](https://store.steampowered.com/app/4336370/) — Survival-Defense Game (Steam, 2026)
Shipped as sole developer under my studio [Pitaya Juice](https://pitaya-juice.itch.io/). ~8,500 lines of C# in Unity 6. Full cycle: design, gameplay systems, AI, UI, Steam certification, launch.

- Custom **A\* pathfinding** over a destructible obstacle graph with NavMesh caching
- Wildlife **AI behaviour trees** with line-of-sight detection and target memory
- Diegetic **contextual prompt/tutorial system** — no pause, no skip
- Full **controller + keyboard support**, UIToolkit UI, JSON-persisted settings

### [ChipChip](https://guillaumepayet.itch.io/chipchip) — CHIP-8/SCHIP/MCHIP Emulator (Multi-platform)
C++ core with strict separation from rendering frontends (SFML, Godot, Unity, Android). Clean architecture, cycle-accurate 60Hz emulation.

- Pure C++ interface boundaries — core knows nothing about rendering backends
- Full **SCHIP/MCHIP opcode extensions** (shift, bitwise, palette, alpha blending)
- Shipped on **Google Play**, Windows, macOS, Linux

> 📦 *The ChipChip core repository is coming soon — watch this space.*

### [Remote Numpad](https://play.google.com/store/apps/details?id=com.guillaumevpayet.remotenumpad) — Android Utility (Open Source)
Android app connecting to a computer via Bluetooth or WiFi to act as a remote numeric keypad. Kotlin client, Kotlin server, Objective-C macOS Bluetooth plugin.
